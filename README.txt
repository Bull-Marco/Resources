Bullacher Ressourcen Readonly Viewer für Netlify

Zweck:
- Externe Handy-Ansicht
- Nur Lesen
- Keine Buchung, kein Löschen, keine Stammdaten
- Lädt die Datei resources.json

So nutzt du es:
1. ZIP entpacken.
2. Bei Netlify als neue Site hochladen.
3. Aus dem internen Ressourcenplaner JSON exportieren.
4. Die exportierte Datei in resources.json umbenennen.
5. resources.json im Netlify-Projekt ersetzen/hochladen.

Dateien:
- index.html
- style.css
- app.js
- logo.jpg
- resources.json

Wichtig:
Die Seite ist nur so aktuell wie der letzte JSON-Export.
